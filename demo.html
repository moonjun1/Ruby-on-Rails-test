🚀 실제 실행 방법

  1. Ruby 설치: RubyInstaller에서 Ruby+Devkit 다운로드
  2. Rails 설치: gem install rails
  3. 의존성 설치: bundle install
  4. DB 설정: rails db:create db:migrate db:seed
  5. 서버 실행: rails server
  6. 접속: http://localhost:3000
